# 条件渲染
## `v-if`
- 指令,需要添加到一个元素上
- 字符串模板`Handlebars`
```html
{{#if ok}}
  <h1>Yes</h1>
{{/if}}   
```
- `Vue`中`v-if`
```html
<h1 v-if="ok">Yes</h1>
```
- `v-else`
```html
<h1 v-if="ok">Yes</h1>
<h1 v-else>No</h1>
```
### template中的`v-if`条件组
- 当切换多个元素的时候,使用`<template>`元素当做包装元素，并在上面使用 `v-if`。最终的渲染结果不会包含 `<template>` 元素。

### v-else
- 使用`v-else`指令来表示`v-if`的"else块"
```html
<div v-if="Math.random() > 0.5">
  Now you see me
</div>
<div v-else>
  Now you don't
</div>
```
- `v-else` 元素必须紧跟在 `v-if` 或者 `v-else-if` 元素的后面——否则它将不会被识别。
